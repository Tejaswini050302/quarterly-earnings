Key Financial Formulas

EPS: \( \displaystyle EPS = \frac{\text{Net Income} - \text{Preferred Dividends}}{\text{Weighted Avg. Shares}} \)

Net Present Value (NPV): \( \displaystyle NPV = \sum_{t=0}^{T} \frac{CF_t}{(1+r)^t} \)

CAGR: \( \displaystyle CAGR = \left(\frac{V_f}{V_i}\right)^{\tfrac{1}{n}} - 1 \)

NPV in Code (Python)

import numpy as np
cashflows = [-2_000_000, 600_000, 650_000, 700_000, 800_000]
r = 0.10  # discount rate
npv = sum(cf / ((1 + r) ** t) for t, cf in enumerate(cashflows))
print(f"NPV: {npv:,.0f}")
